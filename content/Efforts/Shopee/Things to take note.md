- Always need to prioritize based on impact and urgency.
- Impact level
	- High: Can cause financial impact on Shopee (e.g. order drop) or severe user experience (e.g. page cannot load because too slow)
	- Medium: Can cause impact on user experience and also local operator (e.g. Cannot update voucher, slow dispatch time)
	- Low: No immediate impact, but possibility of impact
- Urgency level
	- High: Issue is already happening, will happens in the near future (e.g. Campaign, Big feature release)
	- Medium: There is some project in pipeline that possibility to make it worse
	- Low: No current issue, or changes that can make it worse
	
- If we can connect the **technical impact to business impact** will be good
